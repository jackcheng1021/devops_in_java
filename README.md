# Devops_In_Java
## 1.介绍
- 该项目基于基础云+SaltStack实现远程运维

- 提供运维API

- 实现可视化效果

## 2.应用
- 第一步、将该项目与liberty_openstack相结合，先实现基础云即代码
- 第二步、将该项目与saltstack结合，实现基础设施即代码
- 第三步、将该项目与GitHub结合，实现软件集成即代码
- 第四步、将该项目与kubernetes结合，实现容器云即代码
